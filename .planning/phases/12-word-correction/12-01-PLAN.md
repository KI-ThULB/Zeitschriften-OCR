  <name>POST /save/&lt;stem&gt; — word correction endpoint</name>
  <files>app.py, tests/test_app.py</files>
  <behavior>
    Request body: JSON `{"word_id": "w3", "content": "corrected"}`
    word_id is the positional index string from the ALTO response ("w0", "w1", ..., "wN").

    Success cases:
    - Valid stem + valid word_id + non-empty content → 200 `{"status": "ok"}`; ALTO file on disk has updated CONTENT attribute for that String element
    - ALTO file is not readable (parse failure) → 500 `{"error": "parse failed", "detail": "..."}`

    Validation failure cases (no disk write):
    - Missing or null stem ALTO file → 404 `{"error": "not found", "stem": "<stem>"}`
    - word_id missing from request body → 400 `{"error": "word_id required"}`
    - content missing from request body → 400 `{"error": "content required"}`
    - content is empty string after strip() → 422 `{"error": "Save failed — invalid content"}`
    - word_id index out of range (no Nth String element) → 404 `{"error": "word not found", "word_id": "<word_id>"}`
    - XSD validation fails after substitution → 422 `{"error": "Save failed — invalid content"}`

    Atomic write: write to a tempfile in the same directory, then os.replace() over the original — ensures partial writes never corrupt the file.

    XSD schema: load using pipeline.load_xsd(pipeline.SCHEMA_PATH). If XSD is None (schema file absent), skip validation and write anyway (same behavior as batch pipeline).

    word_id format: "w{N}" where N is a zero-based integer index into the flat list of String elements returned by root.iter(f'{{{ns}}}String'). Same indexing as GET /alto/&lt;stem&gt; response.

    Cases (input → expected output):
    - POST /save/scan_001 `{"word_id": "w0", "content": "Berichtigung"}` → 200 `{"status": "ok"}`
    - POST /save/scan_001 `{"word_id": "w0", "content": ""}` → 422
    - POST /save/scan_001 `{"word_id": "w0", "content": "   "}` → 422 (stripped empty)
    - POST /save/nosuchfile `{"word_id": "w0", "content": "x"}` → 404
    - POST /save/scan_001 `{"word_id": "w9999", "content": "x"}` → 404
    - POST /save/scan_001 `{}` → 400 (missing word_id)
  </behavior>

    ```python
    @app.post('/save/<stem>')
    def save_word(stem):
        if '/' in stem or '..' in stem:
            return jsonify({'error': 'invalid stem'}), 400

        body = request.get_json(silent=True) or {}
        word_id = body.get('word_id')
        content = body.get('content')

        if word_id is None:
            return jsonify({'error': 'word_id required'}), 400
        if content is None:
            return jsonify({'error': 'content required'}), 400
        if not str(content).strip():
            return jsonify({'error': 'Save failed — invalid content'}), 422

        output_dir = Path(app.config['OUTPUT_DIR'])
        alto_path = output_dir / 'alto' / (stem + '.xml')
        if not alto_path.exists():
            return jsonify({'error': 'not found', 'stem': stem}), 404

        try:
            from lxml import etree
            root = etree.parse(str(alto_path)).getroot()
        except Exception as exc:
            return jsonify({'error': 'parse failed', 'detail': str(exc)}), 500

        ns = pipeline.ALTO21_NS
        strings = list(root.iter(f'{{{ns}}}String'))

        # Parse word_id: expect "w{N}" format
        try:
            idx = int(word_id.lstrip('w'))
        except (ValueError, AttributeError):
            return jsonify({'error': 'word not found', 'word_id': word_id}), 404
        if idx < 0 or idx >= len(strings):
            return jsonify({'error': 'word not found', 'word_id': word_id}), 404

        strings[idx].set('CONTENT', str(content))

        # XSD validation gate — serialize to bytes, validate, then write atomically
        out_bytes = etree.tostring(root, xml_declaration=True, encoding='UTF-8', pretty_print=True)

        xsd = pipeline.load_xsd(pipeline.SCHEMA_PATH)
        if xsd is not None:
            try:
                doc = etree.fromstring(out_bytes)
                if not xsd.validate(doc):
                    return jsonify({'error': 'Save failed — invalid content'}), 422
            except Exception:
                return jsonify({'error': 'Save failed — invalid content'}), 422

        # Atomic write: temp file in same directory, then os.replace
        import os, tempfile
        tmp_fd, tmp_path_str = tempfile.mkstemp(
            dir=str(alto_path.parent), suffix='.tmp'
        )
        try:
            with os.fdopen(tmp_fd, 'wb') as fh:
                fh.write(out_bytes)
            os.replace(tmp_path_str, str(alto_path))
        except Exception as exc:
            try:
                os.unlink(tmp_path_str)
            except OSError:
                pass
            return jsonify({'error': 'write failed', 'detail': str(exc)}), 500

        return jsonify({'status': 'ok'}), 200
    ```
