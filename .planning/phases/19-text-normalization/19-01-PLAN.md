```python
last_in_line = set()
for tl in root.findall(f'.//{{{ns}}}TextLine'):

    if tl_strings:
        last_in_line.add(id(tl_strings[-1]))
```

Then in the existing words loop, add `'line_end': id(elem) in last_in_line` to each word dict. The elem variable is already the String element being iterated.

**Step 2 — Build elem_to_idx map (for blocks word_ids):**

After the words loop, rebuild a map from element id() to global index (the index 'i' used to generate word ids like 'w0', 'w1'). This is a second pass over all String elements:

```python
elem_to_idx = {}
for i, elem in enumerate(root.iter(f'{{{ns}}}String')):
    elem_to_idx[id(elem)] = i
```

**Step 3 — Build blocks_out array:**

Iterate TextBlock elements and collect their geometry and word_ids:

```python
blocks_out = []
for block_elem in root.findall(f'.//{{{ns}}}TextBlock'):
    word_ids = [
        f'w{elem_to_idx[id(s)]}'
        for s in block_elem.iter(f'{{{ns}}}String')
        if id(s) in elem_to_idx
    ]
    blocks_out.append({
        'id': block_elem.get('ID', ''),
        'hpos': int(block_elem.get('HPOS', 0)),
        'vpos': int(block_elem.get('VPOS', 0)),
        'width': int(block_elem.get('WIDTH', 0)),
        'height': int(block_elem.get('HEIGHT', 0)),
        'word_ids': word_ids,
    })
```

**Step 4 — Add 'blocks' to the jsonify() return:**

In the existing return jsonify({...}) call, add `'blocks': blocks_out` alongside the existing keys (page_width, page_height, jpeg_width, jpeg_height, words). Do NOT remove any existing keys — this is additive only.

**Namespace:** The existing ns variable in serve_alto() already holds the correct ALTO namespace string (CCS-GmbH). Use it for all new findall/iter calls.
