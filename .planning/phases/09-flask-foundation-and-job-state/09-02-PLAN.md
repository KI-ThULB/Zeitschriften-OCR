  <action>
    Create app.py at the project root (next to pipeline.py). This is a single-file Flask application following the same flat-file convention as pipeline.py. No blueprints, no packages, no create_app() factory — a module-level `app` object.

    **Full structure (implement exactly this, in this order):**

    ```
    Imports (stdlib then third-party)
    Module-level constants (OUTPUT_DIR default, UPLOAD_SUBDIR)
    Module-level state (_job_lock, _jobs, _sse_queue, _run_active)
    app = Flask(__name__)
    Helper: _format_sse(event, data) -> str
    Helper: _ocr_worker(tiff_paths, output_dir, lang, psm, padding) -> None
    Route: POST /upload
    Route: POST /run
    Route: GET /stream
    if __name__ == '__main__': argparse + app.run()
    ```

    **Imports:**
    ```python
    import argparse
    import importlib
    import json
    import queue
    import threading
    import time
    from pathlib import Path

    from flask import Flask, Response, jsonify, request, stream_with_context
    from werkzeug.utils import secure_filename

    import pipeline
    ```

    **Module-level state:**
    ```python
    _job_lock = threading.Lock()
    _jobs: dict = {}          # stem → {filename, state, word_count, elapsed_time, error_msg}
    _sse_queue: queue.Queue | None = None
    _run_active = threading.Event()

    app = Flask(__name__)
    app.config['OUTPUT_DIR'] = './output'   # default; overridden by --output CLI arg
    ```

    **_format_sse:**
    ```python
    def _format_sse(event: str, data: dict) -> str:
        return f"event: {event}\ndata: {json.dumps(data, ensure_ascii=False)}\n\n"
    ```

    **_ocr_worker — critical implementation notes:**
    - Signature: `def _ocr_worker(tiff_paths: list, output_dir: Path, lang: str, psm: int, padding: int) -> None`
    - Iterate tiff_paths sequentially (NOT with ProcessPoolExecutor — macOS spawn issue, no progressive SSE possible)
    - Per-file skip check: `alto_path = output_dir / 'alto' / (stem + '.xml')` — if exists, state='skipped', elapsed=0.0
    - Per-file OCR: call `pipeline.process_tiff(tiff_path, output_dir, lang=lang, psm=psm, padding=padding, no_crop=False, adaptive_threshold=False)` — wrap in try/except Exception
    - On success: read word_count from written ALTO using `pipeline.count_words(root, pipeline.ALTO21_NS)` where root is parsed from the written file
    - On exception: state='error', error_msg=str(e), word_count=None
    - Update _jobs[stem] under _job_lock after each file
    - Post SSE event to _sse_queue after each file (do NOT hold _job_lock while calling .put())
    - After loop: post `_format_sse('run_complete', {'total': len(tiff_paths)})` to _sse_queue
    - In `finally` block: `_run_active.clear()` — MUST be in finally so it clears even if an unexpected exception escapes the per-file try/except

    **POST /upload:**
    - `request.files.getlist('files')` — multi-file field name is 'files'
    - Return 400 if no files provided
    - For each file:
      - Skip if f.filename is empty
      - `filename = secure_filename(f.filename)`
      - Check suffix: `Path(filename).suffix.lower() not in ('.tif', '.tiff')` → append rejected result, continue
      - Normalize stem to LOWERCASE: `stem = Path(filename).stem.lower()` — required so skip check matches pipeline's lowercase output paths
      - Save to `upload_dir / filename` (upload_dir = Path(app.config['OUTPUT_DIR']) / 'uploads')
      - Skip check: `alto_path = Path(app.config['OUTPUT_DIR']) / 'alto' / (stem + '.xml')`
      - Record job: state='done' if already_done else 'pending' (under _job_lock)
      - Append `{'filename': filename, 'status': 'staged', 'already_processed': already_done}` to results
    - Return `{'files': results}`, 200

    **POST /run:**
    - Guard: `if _run_active.is_set(): return jsonify({'error': 'a run is already in progress'}), 409`
    - Collect pending stems under _job_lock: `pending = [stem for stem, job in _jobs.items() if job['state'] == 'pending']`
    - Guard: `if not pending: return jsonify({'error': 'no pending files'}), 400`
    - Set _run_active, reset _sse_queue (under _job_lock: `global _sse_queue; _sse_queue = queue.Queue()`)
    - Capture config values as plain objects BEFORE thread launch (avoid app context issues in thread):
      `output_dir = Path(app.config['OUTPUT_DIR'])`, `lang = app.config.get('LANG', 'deu')`, etc.
    - Build tiff_paths from pending stems and upload_dir
    - Launch `threading.Thread(target=_ocr_worker, args=(...), daemon=True)` and call `.start()`
    - Return `jsonify({'status': 'started', 'files': pending}), 202`

    **GET /stream:**
    ```python
    @app.get('/stream')
    def stream():
        def generate():
            local_q = _sse_queue
            if local_q is None:
                return
            while True:
                try:
                    msg = local_q.get(timeout=30)
                    yield msg
                    if msg.startswith('event: run_complete'):
                        break
                except queue.Empty:
                    yield ': keepalive\n\n'
        return Response(
            stream_with_context(generate()),
            mimetype='text/event-stream',
            headers={'Cache-Control': 'no-cache', 'X-Accel-Buffering': 'no'},
        )
    ```

    **CLI entrypoint:**
    ```python
    if __name__ == '__main__':
        parser = argparse.ArgumentParser(description='Zeitschriften-OCR Web Viewer')
        parser.add_argument('--output', default='./output', help='Output directory (default: ./output)')
        parser.add_argument('--lang', default='deu', help='Tesseract language (default: deu)')
        parser.add_argument('--psm', type=int, default=1, help='Page segmentation mode (default: 1)')
        parser.add_argument('--padding', type=int, default=50, help='Crop padding in pixels (default: 50)')
        parser.add_argument('--port', type=int, default=5000, help='Port (default: 5000)')
        args = parser.parse_args()

        output_dir = Path(args.output)
        output_dir.mkdir(parents=True, exist_ok=True)
        (output_dir / 'uploads').mkdir(exist_ok=True)
        (output_dir / 'alto').mkdir(exist_ok=True)

        app.config['OUTPUT_DIR'] = str(output_dir)
        app.config['LANG'] = args.lang
        app.config['PSM'] = args.psm
        app.config['PADDING'] = args.padding

        app.run(host='0.0.0.0', port=args.port, threaded=True, use_reloader=False)
    ```

    **use_reloader=False** is required — the reloader forks the process and creates a second Flask instance with separate module-level state, breaking the SSE queue.

    **Do NOT call validate_batch() or load_xsd() in this phase.** Validation is a post-Phase-9 concern for the web UI.
