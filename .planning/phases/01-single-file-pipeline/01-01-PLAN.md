**load_tiff() function:**
Signature: `def load_tiff(path: Path) -> tuple[Image.Image, tuple[float, float], list[str]]`
- Opens with `Image.open(path)` (lazy load — do NOT call `.load()` here)
- Checks `getattr(img, 'n_frames', 1)`: if > 1, append `'multi-page TIFF, processed frame 0 only'` to warnings and proceed with frame 0
- Reads DPI via `img.info.get('dpi')`. If None: set `dpi = (300.0, 300.0)`, append `'no DPI tag, using 300'` to warnings
- Returns `(img, dpi, warnings)` where dpi is always a 2-tuple of floats

**detect_crop_box() function:**
Signature: `def detect_crop_box(image: Image.Image, padding: int = 50, min_ratio: float = 0.40, max_ratio: float = 0.98) -> tuple[tuple[int,int,int,int], bool]`

CRITICAL: Use `cv2.THRESH_BINARY + cv2.THRESH_OTSU` — NOT THRESH_BINARY_INV. For archival scans (dark scanner bed, light page), THRESH_BINARY makes the light page content white so it becomes the largest contour. THRESH_BINARY_INV is WRONG here.

Implementation:
1. Get `img_w, img_h = image.size`; compute `orig_area = img_w * img_h`
2. Convert to numpy: `img_array = np.array(image.convert('RGB'))`
3. Grayscale: `gray = cv2.cvtColor(img_array, cv2.COLOR_RGB2GRAY)`
4. Threshold: `_, thresh = cv2.threshold(gray, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)`
5. Find contours: `contours, _ = cv2.findContours(thresh, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)`
6. If no contours: return `(0, 0, img_w, img_h), True`
7. Get bounding rect of largest contour: `x, y, w, h = cv2.boundingRect(max(contours, key=cv2.contourArea))`
8. Compute ratio: `ratio = (w * h) / orig_area`
9. If `not (min_ratio <= ratio <= max_ratio)`: return `(0, 0, img_w, img_h), True`
10. Apply padding with bounds clamping:
    - `left = max(0, x - padding)`
    - `upper = max(0, y - padding)`
    - `right = min(img_w, x + w + padding)`
    - `lower = min(img_h, y + h + padding)`
11. Return `(left, upper, right, lower), False`
