**build_alto21():**
Signature: `def build_alto21(alto_bytes: bytes, crop_box: tuple[int, int, int, int]) -> bytes`

CRITICAL ORDER: Apply crop offset BEFORE namespace rewrite. The offset uses ALTO3_NS to find tags; after string replace, the namespace changes to ALTO21_NS and the tag lookup would fail.

Implementation steps:
1. Parse: `root = etree.fromstring(alto_bytes)`
2. Apply crop offset (only if crop_x != 0 or crop_y != 0):
   - `crop_x, crop_y = crop_box[0], crop_box[1]`
   - `ns_prefix = '{' + ALTO3_NS + '}'`
   - Iterate: `for elem in root.iter():`
   - Get local tag: `local = elem.tag[len(ns_prefix):] if elem.tag.startswith(ns_prefix) else elem.tag`
   - If `local in POSITIONAL_TAGS`:
     - If `'HPOS' in elem.attrib`: `elem.attrib['HPOS'] = str(int(elem.attrib['HPOS']) + crop_x)`
     - If `'VPOS' in elem.attrib`: `elem.attrib['VPOS'] = str(int(elem.attrib['VPOS']) + crop_y)`
   - NEVER touch WIDTH or HEIGHT — only HPOS and VPOS get the offset
3. Serialize: `xml_str = etree.tostring(root, encoding='unicode')`
4. Rewrite namespace: `xml_str = xml_str.replace(ALTO3_NS, ALTO21_NS)`
5. Remove xsi:schemaLocation pointing to ALTO 3 XSD (to avoid contradictory schema reference):
   ```python
   xml_str = xml_str.replace(
       'xsi:schemaLocation="http://www.loc.gov/standards/alto/ns-v3# '
       'http://www.loc.gov/alto/v3/alto-3-0.xsd"',
       ''
   )
   ```
   Also handle the case where the schemaLocation uses a different quote style or minor variation by doing a broader replace targeting `xsi:schemaLocation` if the exact string above is not found (use a regex or simpler heuristic only if needed).
6. Re-parse and serialize with XML declaration:
   ```python
   new_root = etree.fromstring(xml_str.encode())
   return etree.tostring(new_root, xml_declaration=True, encoding='UTF-8', pretty_print=True)
   ```
