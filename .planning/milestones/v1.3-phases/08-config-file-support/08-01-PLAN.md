    load_config(path: Path) -> dict

    Input cases and expected outcomes:

    PASS cases (return dict):
    - Valid JSON, all known keys, correct types → return validated dict with only those keys
    - Valid JSON, empty object {} → return {}
    - Valid JSON, unknown key present → return dict without unknown key; print [WARN: unknown config key 'X'] to stderr
    - Valid JSON, mix of known and unknown keys → return only known keys; warn for each unknown

    EXIT 1 cases:
    - path does not exist → print "Error: config file not found: {path}" to stderr; sys.exit(1)
    - File exists but contains invalid JSON → print "Error: config file contains invalid JSON: {json.JSONDecodeError message}" to stderr; sys.exit(1)
    - Known key with wrong type (e.g. workers="four") → print "Error: config key 'workers' expects int, got str" to stderr; sys.exit(1)
    - int-typed key given a bool value (e.g. workers=true) → reject as type error; print "Error: config key 'workers' expects int, got bool" to stderr; sys.exit(1)

    Type map (CONFIG_TYPES):
      lang: str
      psm: int
      padding: int
      workers: int
      force: bool
      verbose: bool
      dry_run: bool
      validate_only: bool

    CRITICAL: bool is a subclass of int in Python.
      - For int-typed keys: isinstance(value, int) and isinstance(value, bool) → reject (bool masquerading as int)
      - For bool-typed keys: isinstance(value, bool) → accept
      - Correct check for int keys: isinstance(value, int) and not isinstance(value, bool)

    Error message capitalization: "Error:" (capital E, lowercase r) — matches locked decisions verbatim,
    NOT "ERROR:" as used in existing validate_tesseract() — locked decisions take precedence.

      def load_config(path: Path) -> dict:
          if not path.exists():
              print(f"Error: config file not found: {path}", file=sys.stderr)
              sys.exit(1)
          try:
              raw = json.loads(path.read_text(encoding='utf-8'))
          except json.JSONDecodeError as e:
              print(f"Error: config file contains invalid JSON: {e}", file=sys.stderr)
              sys.exit(1)
          validated = {}
          for key, value in raw.items():
              if key not in CONFIG_TYPES:
                  print(f"[WARN: unknown config key '{key}']", file=sys.stderr)
                  continue
              expected = CONFIG_TYPES[key]
              if expected is int:
                  # bool is a subclass of int — reject booleans for int-typed keys
                  if not isinstance(value, int) or isinstance(value, bool):
                      print(
                          f"Error: config key '{key}' expects int, got {type(value).__name__}",
                          file=sys.stderr,
                      )
                      sys.exit(1)
              elif not isinstance(value, expected):
                  print(
                      f"Error: config key '{key}' expects {expected.__name__}, got {type(value).__name__}",
                      file=sys.stderr,
                  )
                  sys.exit(1)
              validated[key] = value
          return validated
