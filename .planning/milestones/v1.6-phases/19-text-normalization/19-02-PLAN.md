**Function 1 — clusterByHpos(blocks, pageWidth):**

Groups column-level blocks (already filtered to non-full-width blocks) by horizontal position bands. A new cluster starts when the gap between adjacent block HPOS values exceeds `pageWidth * 0.05` (5% of page width). Returns an array of cluster objects: `{centerHpos: number, blocks: [...]}` where centerHpos is the mean HPOS of blocks in the cluster.

Algorithm:
1. Sort blocks by hpos ascending.
2. Initialize first cluster with the first block.
3. For each subsequent block: if `block.hpos - lastBlockInCurrentCluster.hpos > pageWidth * 0.05`, start a new cluster. Otherwise, append to current cluster.
4. Compute centerHpos as mean of all block hpos values in the cluster.
5. Return clusters array.

Edge case: if blocks is empty, return [].

**Function 2 — columnSort(words, blocks, pageWidth):**

Sorts words into natural newspaper reading order using TextBlock geometry.

```
FULL_WIDTH_THRESHOLD = 0.6  (blocks wider than 60% of page width AND hpos < pageWidth * 0.05)

1. Separate blocks into fullWidthBlocks and columnBlocks.
2. Sort fullWidthBlocks by vpos ascending (top-to-bottom).
3. Cluster columnBlocks via clusterByHpos(columnBlocks, pageWidth).
4. Sort clusters left-to-right by centerHpos.
5. Within each cluster, sort blocks by vpos ascending.
6. Build wordMap: Object.fromEntries(words.map(w => [w.id, w]))
7. Build ordered ID list: [...fullWidthBlocks.flatMap(b => b.word_ids), ...clusters.flatMap(col => col.blocks.flatMap(b => b.word_ids))]
8. Deduplicate: track seen IDs; append words not in any block at the end (preserves words from ALTO files without TextBlock elements).
9. Map IDs back to word objects, filter out any undefined.

Fallback: if blocks is falsy or blocks.length === 0, return words unchanged (VPOS-order, already document order for single-column files).
```

**Function 3 — rejoinHyphens(words):**

Rejoins end-of-line hyphenated words using the `line_end` flag added by Plan 01.

```javascript
function rejoinHyphens(words) {
  const result = [];
  let i = 0;
  while (i < words.length) {
    const w = words[i];
    if (w.line_end && w.content.endsWith('-') && i + 1 < words.length) {
      const next = words[i + 1];
      // Strip the trailing hyphen and concatenate with the first word of the next line.
      // The joined word inherits the id and coordinates of the first fragment.
      // This preserves click-to-edit behavior: wordById[id] returns the un-rejoined ALTO content.
      const joined = { ...w, content: w.content.slice(0, -1) + next.content };
      result.push(joined);
      i += 2;
    } else {
      result.push(w);
      i += 1;
    }
  }
  return result;
}
```

**Function 4 — normalizeWords(words, blocks, pageWidth):**

```javascript
function normalizeWords(words, blocks, pageWidth) {
  const ordered = columnSort(words, blocks, pageWidth);
  return rejoinHyphens(ordered);
}
```

**Step 2 — applyConfidenceStyling(threshold) function:**

```javascript
function applyConfidenceStyling(threshold) {
  let lowCount = 0;
  document.querySelectorAll('.word[data-wc]').forEach(span => {
    const raw = span.dataset.wc;
    if (raw === '' || raw === null || raw === undefined) {
      // No WC attribute — leave unstyled
      span.style.opacity = '';
      span.title = '';
      return;
    }
    const wc = parseFloat(raw);
    if (!isNaN(wc) && wc < threshold) {
      span.style.opacity = '0.4';
      span.title = `Confidence: ${wc.toFixed(2)}`;
      lowCount++;
    } else {
      span.style.opacity = '';
      span.title = '';
    }
  });
  const badge = document.getElementById('wc-badge');
  if (badge) {
    badge.textContent = lowCount > 0 ? `${lowCount} low-confidence word${lowCount === 1 ? '' : 's'}` : '';
  }
}
```

Note: opacity '0.4' produces ~40% opacity as specified. Exact value is at Claude's discretion; 0.4 is the reference.
